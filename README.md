# VoiceBuddy AI

VoiceBuddy AI is a voice-first AI companion designed for children and elderly users.
It is built using Google AI (Gemini via AI.dev) with a focus on safety, accessibility,
and simple voice interaction.

## Features
- Child Mode: Learning, homework help, stories
- Elder Mode: Reminders, companionship, simple explanations
- Voice-only interaction
- Safety-first AI behavior

## Technology
- Google AI Studio (Gemini 1.5 Flash)
- GitHub

## AI Design
VoiceBuddy uses carefully designed prompts to adapt tone, vocabulary,
and behavior based on the user type (child or elderly).

## Status
Prototype created for a Google AI hackathon.
